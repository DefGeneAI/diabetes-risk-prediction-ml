# Diabetes Risk Prediction using Machine Learning

## Live App
https://diabetes-risk-prediction-ml-bt8nzh2uarnqsqrfc5ntou.streamlit.app/

## Overview
This project predicts diabetes risk using machine learning models trained on the Pima Indians Diabetes dataset.

## Models Tested
- Logistic Regression
- Random Forest
- Gradient Boosting
- Support Vector Machine
- XGBoost

## Best Model
Gradient Boosting
Accuracy: 75.97%
ROC AUC: 0.83

## Tech Stack
Python
Scikit-learn
XGBoost
Pandas
NumPy
Streamlit

- Interactive dashboard
- Real-time predictions
